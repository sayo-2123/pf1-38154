# アプリケーション名m
memolist

# アプリケーション概要
タグ(項目)ごとに共有したメンバーと、メモした内容を共有できる。

# 利用方法

## メモ投稿
1,タグ(項目)を作り、共有するメンバーを指定する  
2,各タグごとにメモ内容(タイトル、内容、画像)を入力し、投稿する。  
3,各タグごとにメモを閲覧、編集、削除できる。
